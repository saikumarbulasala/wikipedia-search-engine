# 🔍 Wikipedia Search Engine
A modern, responsive search interface that fetches real-time data from the Wikipedia API. 

---

## 🚀 Live Demo
**[Click here to try the Live Search Engine](https://saikumarbulasala.github.io/wikipedia-search-engine/)**

---

## 🛠️ Tech Stack & Engineering Focus
* **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox), Bootstrap 4.5
* **Logic:** JavaScript ES6+ (Asynchronous Fetch API, DOM Manipulation)
* **Architecture:** Clean, modular code structure following DRY (Don't Repeat Yourself) principles.

## 🌟 Key Features
* **Asynchronous Search:** Uses `Async/Await` to fetch data without reloading the page.
* **Dynamic UX:** Real-time loading spinner and "Enter to Search" functionality.
* **Performance:** Optimized for speed and minimal API overhead.

## 🏗️ How to Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_GITHUB_USERNAME/wikipedia-search-engine.git`
2. Open `index.html` in your browser.
